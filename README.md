# weui-react-v2

`weui`的`React`实现, 用于微信公众号页面, 包含很多常用组件。[在线例子](https://xueyou2000.github.io/weui-react-v2/site-dest)

## 安装

```bash
# yarn
yarn add weui-react-v2
```

## 开发

```sh
yarn run site-dev
```

## 测试

```
yarn run test
```
