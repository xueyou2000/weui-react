# SwipeAction 滑动操作

滑动操作组件。

## 定义

结合手势操作，从屏幕一侧唤出操作。

## 规则

1.  一次只可滑动一行列表
2.  点击任意按钮之外处或往回滑动该列表可隐藏操作。
